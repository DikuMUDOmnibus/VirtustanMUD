ФОРМАТ ФАЙЛА ОПИСАНИЯ ТРИГГЕРОВ

1) #число - виртуальный номер триггера
2) строка~ - название триггера
3) число флаги число
   1 - класс триггера (см.ниже)
   2 - тип триггера (см.ниже)
   3 - числовой аргумент
4) строка~ - аргумент
5) строка
   ...
   строка
   ~
   - текст триггера


Классы триггеров:

0	MOB_TRIGGER	Тpиггеp для монстpов
1	OBJ_TRIGGER	Тpиггеp для обьектов
2	WLD_TRIGGER	Тpиггеp для комнат

Типы триггеров в зависимости от класса:

* TYPE 0 (mobile) TRIGGERS:
a0	MTRIG_GLOBAL	проверяется даже если зона пуста
b0	MTRIG_RANDOM	проверяется по случайному
c0	MTRIG_COMMAND	когда персонаж ввел команду
d0	MTRIG_SPEECH	когда персонаж сказал фразу
e0	MTRIG_ACT	слово или строка через act
f0	MTRIG_DEATH	персонаж умер
g0	MTRIG_GREET	что-то видимое зашло в комнату
h0	MTRIG_GREET_ALL	что-то зашло в комнату
i0	MTRIG_ENTRY	монстр зашел в комнату
j0	MTRIG_RECEIVE	персонажу дали предмет
k0	MTRIG_FIGHT	каждый пульс во время боя
l0	MTRIG_HITPRCNT	бой и hp меньше некоторого числа
m0	MTRIG_BRIBE	монстру дали деньги
n0	MTRIG_Load	выполняется при загрузке моба 
o0	MTRIG_Memory	выполняется когда моб встречает персонажа из 'памяти' (mremember) 
p0	MTRIG_Damage    выполняется при нанесении по мобу повреждения
z0	MTRIG_AUTO	триггер вызывается из другого триггера

* TYPE 1 (object) TRIGGERS:
*a	OTRIG_GLOBAL     not implemented yet
b0	OTRIG_RANDOM	проверяется по случайному
c0	OTRIG_COMMAND	персонаж ввел команду
g0	OTRIG_GET	предмет подобрали
h0	OTRIG_DROP	персонаж попытался выкинуть предмет
i0	OTRIG_GIVE	персонаж попытался отдать предмет
j0	OTRIG_WEAR	персонаж попытался надеть предмет
l0	OTRIG_REMOVE	пресонаж попытался снять предмет
n0	OTRIG_LOAD	предмет загрузился в игру
o0	OTRIG_UNLOCK	контейнер отперли
p0	OTRIG_OPEN	контейнер открыли
q0	OTRIG_LOCK	контейнер заперли
r0	OTRIG_CLOSE	контейнер заперли
z0	OTRIG_AUTO	триггер вызывается из другого триггера

* TYPE 2 (room) TRIGGERS:
a0	WTRIG_GLOBAL	выполнять даже если зона пуста
b0	WTRIG_RANDOM	проверяется по случайному
c0	WTRIG_COMMAND	персонаж ввел команду
d0	WTRIG_SPEECH	персонаж сказал слово/фразу
f0	WTRIG_RESET	выполняется при репопе зоны
g0	WTRIG_ENTER	персонаж вошел в комнату
h0	WTRIG_DROP	что-то выкинули в комнате
i0	WTRIG_UNLOCK	замок отперли
j0	WTRIG_OPEN	дверь открыли
k0	WTRIG_LOCK	замок заперли
l0	TRIG_CLOSE	дверь закрыли
z0	WTRIG_AUTO	триггер вызывается из другого триггера
