ФОРМАТ ФАЙЛА ОПИСАНИЯ МОБОВ

1) #число - виртуальный номер моба
2) слово слово слово ... слово~ - синонимы, по которым опознается моб
3) строка~ - имя моба в именительном падеже(может быть несколько слов)
4) строка~ - имя моба в родительном падеже
5) строка~ - имя моба в дательном падеже
6) строка~ - имя моба в винительном падеже
7) строка~ - имя моба в творительном падеже
8) строка~ - имя моба в предложном падеже
9) строка~ - описание моба при его нахождении в комнате 
   (например Ворон летает тут)
10)строка~ - описание моба при взгляде на него 
   (например Это небольшая птица с черными перьями)
   Может содержать символы перевода каретки.
11)флаги флаги число E
   1 - флаги моба (см.ниже)
   2 - аффекты, висящие на мобе (см.ниже)
   3 - наклонности (-900 злой - 0 нейтральный - +900 добрый)
   E - обязательный символ 
12) число число число числоdчисло+число числоdчисло+число
    1 - уровень моба
    2 - хитролы моба (-10 - 50)
    3 - AC моба      (10  - -10)
    4-6 - хиты моба (4d5+100 - например)
    7-9 - повреждения наносимые при одном ударе моба (4d4+10 - например)
13) числоdчисло+число число
    1-3 - сколько у моба может быть денег (например 10d5 + 20)
    4   - опыт получаемый при убийстве моба
14) число число число
    1 - позиция при загрузке
    2 - позиция при умолчании
    3 - пол (0 - бесполое, 1 - муж, 2 - жен, 3 - поли)

Далее описание моба продолжается до символа E по принципу параметр: значение
Параметры необязательные, добавляются по необходимости
BareHandAttack: число - тип удара моба (см. ниже) 
Destination: число - виртуальный номер комнаты, куда последует моб после загрузки
Str: число - сила (по умолчанию 11) 
Int: число - интеллект (по умолчанию 11) 
Wis: число - мудрость (по умолчанию 11)
Dex: число - ловкость (по умолчанию 11) 
Con: число - телосложение (по умолчанию 11) 
Cha: число - привлекательность (по умолчанию 11) 

LikeWork: число - шанс того, что моб применит спецумения при битве (сбить, обезоружить и т.п.)
MaxFactor: число - сколько мобов можно убить без замаксивания
ExtraAttack: число - количество дополнительных атак
Class: число - раса моба (см. ниже)
Size: число - размер моба
Height: число - высота моба
Weight: число - вес моба
Race: число - раса моба (см. ниже)
Special_Bitvector: флаги - спецумения моба(см. ниже)
Skill: число число - номер и процент владения умением
Spell: число - номер заклинания кастуемого
Helper: число - виртуальный номер моба, которого призовет 
        на помощь этот моб при битве
Строк Skill, Spell и Helper может быть несколько
E - обязательное завершение моба
T - необязательная строка или несоклько - привязывает к мобу тригера

Флаги мобов(строка 11 флаг 1)

MOB_AGGR_NIGHT   (1 << 9)       нападает только ночью
MOB_AGGR_FULLMOON (1 << 10)     нападает в полнолуние
MOB_MEMORY        (1 << 11)     запоминает напавших
MOB_HELPER     (1 << 12)        помогает мобам при битве
MOB_NOCHARM     (1 << 13)       нельзя зачармить

MOB_NOSLEEP      (1 << 15)      нельзя усыпить

MOB_NOHOLD       (1 << 19)      нельзя захолдить
MOB_NOSIELENCE   (1 << 20)      нельзя замолчать
MOB_QUESTGOOD    (1 << 21)      в квесте для добрых (потом поменяется)
MOB_QUESTEVIL    (1 << 22)      в квесте для злых (потом поменяется)
MOB_NOFEAR       (1 << 23)      не убежит при кастинге страха

MOB_AGGR_SUMMER  (INT_ONE | (1 << 5)) моб агрессивен летом
MOB_AGGR_AUTUMN  (INT_ONE | (1 << 6)) моб агрессивен осенью
MOB_LIKE_DAY     (INT_ONE | (1 << 7)) моб появится в зоне только днем
MOB_LIKE_NIGHT   (INT_ONE | (1 << 8)) моб появится в зоне только ночью
MOB_LIKE_FULLMOON (INT_ONE | (1 << 9)) моб появится в зоне только в полнолуние
MOB_LIKE_WINTER   (INT_ONE | (1 << 10)) моб появится только зимой
MOB_LIKE_SPRING   (INT_ONE | (1 << 11)) моб появится только весной
MOB_LIKE_SUMMER   (INT_ONE | (1 << 12)) моб появится только летом
